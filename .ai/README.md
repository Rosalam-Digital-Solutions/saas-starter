# AI Workspace

This folder is project memory for AI pair-programming in this repository.

Future AI agents should read `.ai/PROJECT_CONTEXT.md` first, then `.ai/ARCHITECTURE.md`, `.ai/CODING_RULES.md`, and `.ai/TASKS.md`.

The purpose of this folder is to reduce repeated explanations and context loss between sessions.

Any major architectural change should be recorded in `.ai/DECISIONS.md`.
Any new system area should get its own `.ai/*.md` document.
